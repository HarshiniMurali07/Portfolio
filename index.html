Projects

1. Exploratory Data Analysis (EDA)

This project focused on performing exploratory data analysis (EDA) on a complex dataset. Key tasks included analyzing distributions, identifying outliers, and uncovering trends. Visualizations such as histograms and scatter plots provided insights into the underlying patterns. EDA helped in transforming raw data into a cleaned format suitable for machine learning.

2. Station Lines Visualization

This project utilized data visualization techniques to plot multi-line graphs representing station lines. Each line was styled with unique colors, making patterns easier to interpret. This project demonstrated advanced use of visualization libraries such as Matplotlib and Seaborn to make data more intuitive and actionable.

3. Label Encoding for Categorical Data

This project focused on preprocessing categorical data using label encoding. The aim was to convert non-numeric data into numerical values, ensuring compatibility with machine learning algorithms. Visualizations highlighted the distribution of categories before and after encoding, showcasing its impact on model performance.

4. Product Recommendation System

Built a product recommendation system using collaborative filtering techniques. The model utilized user-product interaction data to suggest highly relevant items to individual users. The recommendation accuracy and system performance were evaluated using metrics like precision and recall, visualized through bar and heatmap charts.

5. Missing Data Analysis

This project focused on analyzing and handling missing data. The analysis included identifying features with missing values and applying imputation techniques such as mean, median, and mode. Missing data patterns were visualized to assess their impact on the dataset's integrity and downstream machine learning workflows.
